# campus-life-super-app

# Project Description
The Campus Life Super App is a multi-page web application designed to improve the college student experience by providing essential campus resources in one centralized platform. This app helps students easily find campus events, explore dining options, and locate study spaces using an interactive map.

The goal of this project is to combine multiple web development skills into a functional, user-friendly, and responsive application that solves campus life challenges.

# Features
*Home Page*
Navigation to all major features
Clean and simple user interface

*Events page*
Attempted to use a Ticketmaster API

*Dining Page*
Shows available dining options
Allows users to save a favorite dining location using localstorage
*Map/Study Spots Page*
Interactive map using Leaflet API
Displays key campus locations with markers
Allows user to view study spots

# Technologies Used
